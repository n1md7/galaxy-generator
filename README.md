### Running

```bash
# Install dependencies
npm install
# For pre-commit hooks, run once
npm install husky

# Build
npm run build:github # For GitHub Pages
npm run build:crazy # For Crazy Games
npm run build:itch # For Itch.io

# Run development server
npm run dev

# Run preview server
npm run preview

# format code
npm run format
```
